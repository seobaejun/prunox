카페트클리닝

홈 / 서비스 / 카페트클리닝

서비스 소개

카펫은 섬유(패브릭) 소재로 만들어져 있기 때문에 세균과 진드기가 번식하기 아주 좋은 환경입니다. 특히 바닥에 깔려있는 카펫의 경우 세탁이 어렵기 때문에 깨끗한 관리가 어렵고 장마철, 겨울철 비와 눈으로 젖은 카펫을 방치하고 건조되게 되면 심각한 오염이 반복되기 때문에 정기적인 클리닝이 필요합니다.

카펫 전문 클리닝 장비를 사용하여 카펫을 세척한 것과 같이 완벽한 클리닝을 도와드립니다.

카펫클리닝 서비스 범위

1차 청소

  • ✓ 패브릿 전용 청소기로 먼지와 진드기 흡입
  • ✓ 카펫의 껌, 오염, 찌든 때 제거
  • ✓ 찌든 때 제거를 위한 전처리 약품 도포
  • ✓ 핸드 폴리싱 작업을 통해 찌든 때 제거
  • ✓ 고온의 스팀으로 살균 및 오염 제거

2차 청소

  • ✓ 버핑머신(브러쉬패드,본넷패드)으로 샴푸 작업
  • ✓ 섬유 유연제 헹굼으로 섬유 손상 복구
  • ✓ 정수된 물을 분사함과 동시에 오염물 흡입
  • ✓ 건조작업을 통한 세균 번식 억제
  • ✓ 주변 정리 및 청소 작업
