준공청소

홈 / 서비스 / 준공청소

서비스 안내

준공청소는 건물이 완공된 후에, 사용자들이 입주하거나 사용하기 전에 필요한 청소 작업으로 공사 후에 남아있는 건축폐기물과 먼지, 페인트 자국 등 건축 시 사용된 공사 잔재들을 깨끗하게 청소합니다. 준공 청소를 통해 건물에 입주하기 전 건물을 청소하게 되면 건물에 대한 깨끗한 이미지를 만들고 신축 건물 입주 시에 쾌적한 환경에서 입주가 가능한 상태로 만들어드립니다.

준공청소 서비스 범위

건물 내부 청소

  • ✓ 보양지, 스티커, 잔여 쓰레기 제거
  • ✓ 공사 후 남은 먼지, 시멘트, 페인트 등 제거
  • ✓ 출입문, 유리창, 창틀, 샷시 청소
  • ✓ 건물의 바닥, 계단, 복도 청소
  • ✓ 벽면, 천장, 조명 먼지 및 오염 제거
  • ✓ 건물 각 세대 내부 청소

공용시설 청소

  • ✓ 로비, 우편함, 커뮤니티 시설 청소
  • ✓ 승강기, 에스컬레이터 내·외부 청소
  • ✓ 화장실 내부 변기, 세면대 등 청소

냉난방 시설 청소

  • ✓ 에어컨, 냉난방기 분해 청소
  • ✓ 공조기, 환기, 정화시스템 청소
  • ✓ 에어 필터 교체, 전열교환기 청소

건물 외부 청소

  • ✓ 건물 외부 유리창, 샷시 청소
  • ✓ 건물 외벽 마감재, 외장재 청소
  • ✓ 잔여 공사 자재 및 폐기물 청소
  • ✓ 건물 주변 쓰레기 및 오염 제거
